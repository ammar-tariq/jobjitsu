# JobJitsu Success Messages

Success is a **quiet nod**. Acknowledge the technique; don’t throw a parade for routine saves.

---

## Principles

1. **Measured warmth.** Jade check + short line beats confetti.
2. **Count craft, not vanity.** Weekly throws are fine; “streaks” and leaderboards are not.
3. **Metaphor sparingly.** Best on meaningful milestones (application sent), not every checkbox.
4. **Confirm control.** Especially after send — the user chose this.
5. **Point forward only when useful.** One optional next step max.

---

## Anatomy

```
Title — what succeeded
Optional body — context or gentle count
Optional action — View application / Schedule follow-up
```

Toasts auto-dismiss; inline confirmations can persist near the object.

---

## Patterns

### Application sent
**Title:** Application sent  
**Body:** One more throw. That’s {n} this week.  
**Action:** View application · Set follow-up  

### Draft saved
**Title:** Draft saved  
**Body:** Ready when you are.  
**Action:** none  

### Queued for review
**Title:** Queued for review  
**Body:** Tailored and waiting for your approval.  
**Action:** Review  

### Preferences updated
**Title:** Preferences saved  
**Body:** The agent will follow these on the next run.  
**Action:** none  

### Follow-up sent
**Title:** Follow-up sent  
**Body:** A polite nudge is on its way.  
**Action:** View timeline  

### On-device model connected
**Title:** On-device model connected  
**Body:** Ready in this dojo. No cloud required.  
**Action:** none  

### Onboarding complete
**Title:** You’re set  
**Body:** Resume connected. Preferences saved. Agent ready for your signal.  
**Action:** Open queue  

### Resume imported
**Title:** Resume added  
**Body:** Stored on this device for tailoring and review.  
**Action:** Continue  

---

## Milestone vs Routine

| Routine (plain) | Milestone (light metaphor OK) |
|-----------------|-------------------------------|
| Draft saved | First application sent |
| Preferences saved | Onboarding complete |
| Row reordered | Weekly throw count (soft) |

If unsure, choose plain.

---

## Visual & Motion

- Jade (`#10B981`) + check icon
- Toast rise 240–320 ms; reduced motion → instant
- No full-screen overlays for standard success
- Sound: off by default

---

## Don’ts

| Don’t | Do |
|-------|-----|
| “YOU CRUSHED IT!!!” | “Application sent.” |
| Confetti for saving a field | Quiet toast |
| Fake social proof | Personal count only (“3 this week”) |
| Imply an interview is guaranteed | Stop at send / follow-up truth |

---

## Pairing with Voice

Canonical send line from brand:

> “One more throw. That’s {n} this week.”

Keep `{n}` accurate. If count unavailable, drop the second sentence rather than inventing one.
