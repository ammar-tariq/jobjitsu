# Brand assets

Canonical JobJitsu marks from [docs/brand/BRAND_GUIDELINES.md](../docs/brand/BRAND_GUIDELINES.md).

| File | Usage |
|------|--------|
| `logo-icon.svg` | App icon, favicon, navbar |
| `logo-full-horizontal.svg` | README / marketing wordmark |
| `logo-stacked.svg` | Splash / social profiles |
| `logo-icon-mono-dark.svg` | Dark backgrounds (white mark) |
| `logo-icon-mono-light.svg` | Light backgrounds (indigo mark) |
| `social-preview.svg` | Open Graph / Twitter card |

Teal (`#2DD4BF`) is reserved for the envelope flap accent only on the full-color icon.
