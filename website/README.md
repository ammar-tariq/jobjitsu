# @jobjitsu/website

Docusaurus 3 documentation site for JobJitsu.

## Source of truth

Markdown is **not** copied into this package. The site reads the monorepo [`/docs`](../docs) directory in place (`docusaurus.config.ts` → `docs.path`).

## Commands

```bash
# from repo root
pnpm --filter @jobjitsu/website dev
pnpm --filter @jobjitsu/website build
pnpm --filter @jobjitsu/website serve
```

## Features

- Dark mode default (Midnight Ink + Electric Teal)
- Inter as the primary UI font
- Mermaid diagrams (`@docusaurus/theme-mermaid`)
- Prism syntax highlighting
- Autogenerated sidebar from `/docs` folders
- GitHub link in the navbar
