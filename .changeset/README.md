# Changesets

Versioning and changelog tooling for `@jobjitsu/*` packages.

```bash
pnpm changeset          # record a change
pnpm version-packages   # bump versions + changelogs
pnpm release            # build + publish (when packages are public)
```

Packages are private for now (`access: restricted`). See [Changesets docs](https://github.com/changesets/changesets).
